Preview status and transcoding failure

Time: ~2 minutes Device: Web app

Confirm your file is supported and contact us with details so we can resolve the transcoding failure quickly.

Checklist

Follow the steps below to complete this task.

What you need
  • Smart VR Lab account at app.smartvrlab.nl
  • Stable internet connection
  • Device access as noted above
Outcome

Complete these steps and you will be ready for the next publishing action.

Step 1 - Check file support

Confirm your media matches our supported filetypes (codec, resolution, and projection).

If it is not supported, convert the file and upload again.

Step 2 - Retry the upload

If the file is supported, re-upload the source file and wait for transcoding to finish.

Keep the browser open until the status updates.

Step 3 - Contact us

If the issue persists, report it via our Transcoding failure form.

Include the filename, size, format, and what you expected versus what happened.

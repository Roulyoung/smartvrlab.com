Helpdesk Content / Library

Change 2D / 3D status and titles

Time: ~3 minutes Device: Web app

Update 2D/3D settings, layout, and titles for existing videos or images.

Jump to steps Back to helpdesk

Checklist

Follow the steps below to complete this task.

What you need
  • Smart VR Lab account at app.smartvrlab.nl
  • Stable internet connection
  • Device access as noted above
Outcome

Complete these steps and you will be ready for the next publishing action.

Step 1 - Open the item

In Content, click the item you want to edit.

Step 2 - Adjust format

Set 2D or 3D and choose the correct layout if 3D.

Step 3 - Edit metadata

Update the title and description to reflect the correct format.

Save your changes.

Step 4 - Republish if needed

If the item is in a playlist already published, republish that playlist to push the new format.

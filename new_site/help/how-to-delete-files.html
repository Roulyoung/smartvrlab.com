How to delete files

Time: ~2 minutes Device: Web app

Remove old or unused media to keep your library clean and make room for updated content.

Jump to steps Back to helpdesk
What you need
  • Access to app.smartvrlab.nl
  • Permission to manage content
  • List of items to remove
Outcome

Unused files are removed and storage stays organized.

Step 1 - Open Content

Go to Content and find the item (video or image) you want to remove.

Step 2 - Check usage

Confirm the file is not needed in playlists or interactives. If it is, replace it first.

Step 3 - Delete the file

Use the delete/remove option for the item. Confirm the prompt to finalize deletion.

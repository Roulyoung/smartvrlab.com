Case study

Amsterdam Medical Centre

Amsterdam University Medical Centre is one of the top leading Dutch institution when it comes to training psychology, pharmacy and neurology students. They looked for a way to streamline their brain-anatomy course to not only make it more accessible, but to also create a more realistic learning experience.

After some literal brainstorming, together we came up with a 3D application that provides a map with all structures of the brain. It not only displays the anatomy in great detail, it also include a quiz mode for students to rehearse and test themselves.

  • Web app accessible through all browsers
  • Contains over 140 brain structures
  • Interactive quiz mode to grade students
  • Worldwide used by universities
